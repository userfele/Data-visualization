# Data Visualization

A collection of Java programs that parse CSV files and render charts using the XChart library.

---

## Charts Included
- **World Population (1950-2020)**
- **US Unemployment Rate (1948-2020)**
- **COVID-19 Cases: USA vs EU**
- **Fahrenheit to Celsius Conversion**
- **Predator vs Prey Populations**

## Libraries Used
- knowm.xchart (XChart)

## How to Run
Compile and run any of the chart files in an IDE that includes the XChart library and the required CSV file in the resources/ folder.
